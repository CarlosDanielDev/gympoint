# Gympoint

Gympoint- Serviços de gerenciamento de academias.

## Instalação

Use o git clone [git](https://github.com/DanPHP7/gympoint/) para clonar a Gympoint.

```bash
git clone https://github.com/DanPHP7/gympoint.git
```

## Run

```bash
yarn && yarn dev
```
